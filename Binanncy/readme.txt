WordPress plugin to connect binance and 3commas for user accounts.

== Description ==

The <strong>Binanncy</strong> plugin allows users to quickly add there API data for Market Vision trading.


== Installation ==

Easy wordpress standard installation.
<p>Simply upload binanncy.zip and activate plugin</p>
<p><strong>Available Shortcodes</strong></p>
<li>[binanncy] - The user front end dashboard</li>
<li>[binanncy_settings] - The menu buttons for user (must be on same page)</li>
